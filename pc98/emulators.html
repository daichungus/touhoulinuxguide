List of known PC-98 emulators

For more information, check out this list.

DOSBox-X

All-around good choice for most users.

Natively runs on Linux, Windows, and macOS.

Under active development. Open source.

  • Official website
  • Flathub page
  • Fedora Copr page

Neko Project II

The most popular family of PC-98 emulators due to their relatively accurate emulation.

Neko Project II kai

Recommended for Linux and RetroArch users.

Natively runs on Linux (and multiple platforms as a RetroArch core)

Under active development. Open source.

  • Github repo (source code and build instructions)

  • Libretro Docs page*

* This is the documentation, not the install page! To install the core, do so in RetroArch!

Neko Project 21/W

Recommended for Windows users.

Natively runs on Windows (XP and later).

Under active development. Open source.

  • Download link page (choose the one with "通常版です。ソースコード付きです。")
  • Github repo

Neko Project II fmgen

Recommended for Windows users.

Natively runs on Windows.

Under active development. Closed source.

  • Download link (choose the most recent version of np2fmgen.7z)

Neko Project II

Obsolete. Not recommended.

Natively runs on Windows and Mac (older Intel).

No longer under active development. Open source.

  • Download link
